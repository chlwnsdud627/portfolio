# Naming Convention
- BEM 표기법

# Commit Convention
- Feat: 새로운 기능을 추가 
- Fix: 버그 수정
- Design: CSS 등 디자인 변경
- Docs: 문서 수정
- Style: 코드 수정과는 별개로, 코드 포맷 변경 또는 세미콜론 누락등을 수정
- Refactor: 코드 리팩토링
- Rename: 파일 혹은 폴더명 수정한 경우
- Remove: 파일 삭제작업만 수행한 경우
- Comment: 필요한 주석 추가
